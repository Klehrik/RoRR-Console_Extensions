Adds a bunch of useful commands to the console.  

Type `help (command)` to view usage.  
- `count`
- `god`
- `gold`
- `item_give`
- `item_take`
- `kill_all`
- `level`
- `locate`
- `lock_all`
- `seppuku`
- `skip_teleporter`
- `spawn`
- `spawn_item`
- `spawn_monster`
- `stage_goto`
- `toggle_spawning`
- `unlock_all`
- `zoom`
